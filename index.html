Welcome to the bear's cave

Hello! This is my personal webpage where I host my (mostly dead) projects and other cool stuff about me. I will keep these pages as minimal and resource-friendly as possible, without anything else that web doesn't needs to work. Here you can find out who I am or what I do as well, and some config files for Linux/Windows ricing, scripts, and other things I like to code or do. I also make some music under the VIRTUAL PLEASURE artistic name and I'm trying to upload them here from time to time.

As I don't use almost any social network pages to share content I tought it was a good idea to create this page and share everything I would like if I had one, with no restrictions or censorship. Even if I don't update it too often, I like how it looks. If you want to know more stuff about me and my character check under bio and fursona, I have a pretty cool art gallery of my sona there if you want to sneak into it. And I usually make some blog posts and a few Linux tutorials on how to fix/run some weird stuff if you're into that kind of stuff.

Just a quickie about the styling: literally all CSS code on these pages are embedded on the HTML code from the pages itself. I just find that more practical and easier to make some adjustments, so I don't have a lot of separate CSS files to a big number of pages. And there's nothing special about the CSS, just plain and simple formatting!
